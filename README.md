# rofi-wallpaper
simple wallpaper menu for rofi with multi-monitor support, aka it will ask you to select a display first if you have more than one

dont forget to install [nitrogen](https://github.com/l3ib/nitrogen) or modify the script for whatever you use for wallpapers

![example](https://raw.githubusercontent.com/antoine-was-unavailable/rofi-wallpaper/refs/heads/main/demo.gif)

the script takes 3 arguments in the following order:
- 1: path to your wallpaper folder
- 2: path to the rofi config file used for the monitor selection
- 3: path to the rofi config file used for the wallpaper selection
